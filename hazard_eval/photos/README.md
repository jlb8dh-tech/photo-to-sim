# hazard_eval/photos

Facility photos used as the fixed test set for the hazard inspection eval harness.

## Coverage

5 photos per industry:
- **manufacturing** 
- **chemical** 
- **warehousing** 
- **construction** 
- **food processing** 

Mix of clean/messy, wide/close-up, bright/dim lighting, clear/blur per the spec.

Re-runnable; existing files are skipped. Prints a per-file `ok / skip / FAIL` line and a summary.


**Important: lock this set after baseline.** Per the brief, the rubric and test set should be frozen in Week 2 so Week 6 numbers are comparable.
